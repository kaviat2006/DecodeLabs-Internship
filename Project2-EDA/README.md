# Exploratory Data Analysis (EDA) - Project 2

## Project Overview
This project is part of the DecodeLabs Data Analytics Internship. The objective is to perform Exploratory Data Analysis (EDA) on the given dataset to identify patterns, trends, outliers, and generate meaningful insights.

## Dataset
- Dataset Name: Dataset for Data Analytics (1).xlsx
- Total Records: 1200
- Features:
  - OrderID
  - Date
  - CustomerID
  - Product
  - Quantity
  - UnitPrice
  - ShippingAddress
  - PaymentMethod
  - OrderStatus
  - TrackingNumber
  - ItemsInCart
  - CouponCode
  - ReferralSource
  - TotalPrice

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Exploratory Data Analysis Performed
- Loaded the dataset
- Checked data types and missing values
- Calculated descriptive statistics
- Analyzed product sales
- Analyzed payment methods
- Analyzed monthly sales
- Identified outliers using a boxplot
- Created visualizations

## Visualizations
- Bar Chart
- Pie Chart
- Histogram
- Line Chart
- Boxplot

## Key Findings
- Identified the most sold products.
- Analyzed customer payment preferences.
- Observed monthly sales trends.
- Detected outliers in TotalPrice.

## Conclusion
The dataset was successfully analyzed using Exploratory Data Analysis (EDA). Statistical analysis and visualizations provided useful insights into customer orders, sales trends, and purchasing behavior.
